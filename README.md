# cDCGAN WebApp

## 使い方

学習済みのconditionalDCGANで, MNIST画像を生成します.

ラベルを1に指定するとそのラベルの画像が生成されます.

各ラベルは0から1まで指定できます. 中間的な表現も作れるので試してみてください!!
